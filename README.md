# Personal Portfolio

This project is a simple personal portfolio website for Llana Pacis. It highlights who she is, what skills she is learning, her school projects, hobbies, education, and a contact form.

## What’s included
- A welcoming home section with a short introduction
- A skills section for technologies and tools
- A projects section featuring selected work
- A hobbies section showing interests outside school
- An education and experience timeline
- A contact form for reaching out

## Structure
- index.html: main page structure
- css/style.css: styling for the website
- js/script.js: dynamic content loading
- data/data.json: content for skills, projects, hobbies, and timeline

## How to view it
Open index.html in your browser to view the portfolio locally.
